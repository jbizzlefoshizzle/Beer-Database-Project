Summary

Visualizations

The purpose of this project was to see if the weather gets warmer as you get closer to the equator. To accomplish this, I pulled data from the OpenWeatherMap API to create a database with information from over 500 randomly chosen cities.

Matplotlib was used to plot various aspects of weather against latitude. We looked at temperature, cloudiness, wind-speed and humidity.
